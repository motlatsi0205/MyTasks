# My Tasks - Android Application

A modern, user-friendly task management application built with Kotlin and Material Design.

## Features
- **Add Tasks**: Easily enter task descriptions.
- **Schedule**: Set specific dates and times for your tasks using native pickers.
- **Smart Sorting**: Tasks are automatically sorted so the nearest upcoming task is always at the top.
- **Auto-Cleanup**: Tasks are automatically removed once their scheduled time has passed.
- **Persistence**: Tasks are saved locally on the device, so they aren't lost when the app is closed.
- **Modern UI**: Built with Material Design components, featuring rounded cards and a clean interface.

## Technologies Used
- **Kotlin**: Primary programming language.
- **Material Design**: For a modern UI/UX.
- **SharedPreferences & JSON**: For local data persistence.
- **Handler/Runnable**: For real-time background task monitoring.

## How to Run
1. Clone this repository.
2. Open the project in **Android Studio**.
3. Build and run on an emulator or a physical Android device.
